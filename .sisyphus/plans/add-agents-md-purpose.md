### 预期内容
```markdown
## 作用与用途

`AGENTS.md` 是 **AI 智能体开发指南**，旨在为 AI Agent 提供以下信息：

| 用途 | 说明 |
|------|------|
| **项目认知** | 帮助 Agent 快速了解项目技术栈、架构和目录结构 |
| **开发规范** | 提供代码风格、命名规范、禁止模式等开发准则 |
| **构建指南** | 列出构建、测试、运行的常用命令 |
| **架构指导** | 说明项目采用的架构模式（FlowMVI、MVVM 等）|
| **减少询问** | 涵盖常见开发问题的解决方案，减少 Agent 向用户提问的次数 |

> **注意**：`AGENTS.md` 存在于项目根目录及多个子目录中，每个文件仅描述其所在目录的模块信息。根目录的 AGENTS.md 包含全局性规范，子目录的 AGENTS.md 描述特定模块。
```
